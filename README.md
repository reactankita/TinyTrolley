TinyTrolley is a modern grocery management app built with React Native and Firebase.
It allows users to sign up, log in, add grocery items with quantities, automatically fetch approximate prices using an open-source product API, and calculate total cart cost intelligently.

Features:
-Firebase Authentication
-Add, view, and delete grocery items
-Dynamic cart with total price calculation
-Smart search and unit-based quantity handling
-Cloud Firestore database

Tech Stack:
Frontend: React Native, React Navigation, React Native Paper
Backend: Firebase Authentication, Firebase Firestore, Product Price API
